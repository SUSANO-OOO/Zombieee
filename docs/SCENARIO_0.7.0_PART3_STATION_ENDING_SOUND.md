# サウンド正本

## 全体

- BGMは常に鳴らし続けない。無音を恐怖、喪失、判断の間として使う。
- 会話中はBGMを3〜5dB下げ、重要台詞時は環境音も1〜3dB下げる。
- 台詞終了後250〜500msで復帰。
- スキップ後、BGM・環境音・ループSEが残留しない。
- スマホ本体スピーカーで台詞が聞き取れるよう、中低域の濁りを避ける。
- イヤホンでは環境音の定位を広げすぎない。疲れるため。
- 重要警告は戦闘SEより優先。

## 導入

- Scene 0-1はBGMなし。空調、蛍光灯、ケージ音だけ。
- 西新の日常は軽い雑踏とベース。崩壊に近づくにつれて低音が濁る。
- ユウタの通信切断後は一瞬だけ音を引く。
- タイトル直前は2秒完全無音。

## CRAWLER

- エンジン、食器、工具、生活音を薄く鳴らす。
- 食堂会話はBGMより人の声と食器音を前へ出す。
- 通信受信時は環境音を少し狭める。

## Stage 1〜3

- 商店街は雨、アーケード、遠い呻き。
- 早良区役所は不調エンジンと避難者の息遣い。
- TAKUYA戦はクマバーソンが写真を見る間だけ音数を減らす。
- TAKUYA停止後は勝利ファンファーレにしない。短く重い余韻。

## 西新駅地下

- 改札区域はBGMなし開始。換気、漏水、金属音で圧迫感を出す。
- ホームは機械的リズム。台車音をBGMの一部のように扱う。
- 保守トンネルは心拍に近い低音。封鎖扉と電源盤のSEを戦闘進行の合図にする。
- 改札喰いは巨大咆哮ではなく、金属と肉が擦れる低い音。台詞を邪魔しない。

## ボイス

- 出撃、危機、撤退、特殊敵反応を短く収録する。
- 口癖を毎回使わない。
- パイセンは勢いと恐怖が同居する。
- クマバーソンは危機時ほど声が低く落ち着く。
- ババヤガは奇妙な言葉を一場面に一つ程度。
- ノイズは叫ばない。声の硬さで緊急度を出す。
