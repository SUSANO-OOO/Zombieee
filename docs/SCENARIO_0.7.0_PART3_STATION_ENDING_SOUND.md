# 3. サウンド実装正本

## 3.1 必須カテゴリ

- BGM
- 環境音
- 物語に関係するSE
- UI・操作SE
- 戦闘SE
- 短いキャラクターボイス
- 無音区間
- ボイスダッキング

## 3.2 導入

| 区間 | BGM | 環境音・SE | 演出 |
|---|---|---|---|
| 研究棟 | なし | 空調、蛍光灯、ケージ、隔離扉、紙コップ | 死後反応のラットを音で先に見せる |
| 救急搬入口 | なし | 救急車、首輪、犬の唸り | 咬傷を映さず音で切る |
| 西新の平常 | 街へ溶ける軽い曲 | 店、瓶、雨、地下鉄振動 | 徐々に不協和音 |
| 発生 | BGM停止 | クラクション、悲鳴、ガラス、火災報知器 | 音の密度を上げた後、停電で切る |
| タイトル | 短い専用曲 | 壊れた店内放送の残響 | 2秒無音後に開始 |

## 3.3 会話

- 会話開始時、BGMを3〜5dB下げる。
- 重要台詞時、環境音を1〜3dB下げる。
- 台詞終了後250〜500msで戻す。
- スキップ後にBGMや環境音ループを残さない。
- 戦闘中警告は、BGMよりも戦闘SEよりも優先する。

## 3.4 スマートフォン聴感

- 本体スピーカー、イヤホン、PCで確認。
- 小音量でも警告と台詞が聞こえること。
- BGMが良くても台詞を邪魔する場合はBGM側を下げる。
- AudioContext解除、復帰、二重再生、無音復帰を検証する。
