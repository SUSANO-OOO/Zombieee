# STAGE 2　早良区役所

## event: `stage-sawara-pre-v5`

環境音：アイドリング不良、風で鳴る案内板、遠い呻き声、泣き疲れた子どもの寝息。  
BGM：`bgm-sawara-before-defense`。低いテンポ。車両の不調音を邪魔しない。

[クマバーソンが弁当を配る。避難民の女性が受け取らない。センセイが負傷者の足を固定している。]

**クマバーソン**「足りん時は俺が食わん。だから食え」

[女性が弁当を受け取る。少しだけ頭を下げる。]

**センセイ**「クマバーソン、あとであなたも食べる」

**クマバーソン**「食べた」

**センセイ**「嘘」

**センセイ**「入らない」

**ロッカ**「今その比較いる？」

**ババヤガ**「緊張を共有しています」

[庁舎内から白衣の男が出てくる。ケースを胸に抱えている。避難民の列には並ばない。]

**白衣の男**「その車両に乗せてください。重要資料です」

**センセイ**「怪我は」

**白衣の男**「ありません」

**センセイ**「じゃあ最後」

**白衣の男**「これは、人類に関わる資料です」

**パイセン**「目の前の人類から乗せるんじゃないすか」

[白衣の男、黙る。ケースを抱える手だけ強くなる。]

**ババヤガ**「ケースを預かります。あなたは担架」

**白衣の男**「触らないでください！」

[避難民の子どもが起きる。全員が男を見る。]

**クマバーソン**「声でかい。腹減っとる時の声や」

**白衣の男**「違います」

**クマバーソン**「じゃあ怖い時の声や。どっちでも食え」

[白衣の男は返事できない。]

**ノイズ（無線）**「東側から群れ。車両の音に寄ってる」

**ロッカ**「ほら、私の音に文句言う客が来た」

**パイセン**「接客しなくていいっす」
