更新日：2026-07-17  
script version：`outbreak-origin-v8`

## 1. 実装契約

- プレイヤーは無言の指揮官。会話選択肢、性格分岐、固定台詞を追加しない。
- 指揮官の判断は、既存の出撃決定、マーカー、総攻撃、帰還操作と、固定カット内の視線・手振りで示す。
- 戦闘中barkは、該当人物が実際に出撃中の場合だけ表示する。
- Stage 2とStage 3の既存目的、敵構成、勝利条件を変更しない。
- ストーリー会話の読み上げボイスは使用しない。台詞は画面表示として扱い、既存の戦闘中キャラクターボイスは維持する。
- `医療支援`は正式表示名決定後に一括置換する役割ラベルであり、`センセイ`は使用しない。
