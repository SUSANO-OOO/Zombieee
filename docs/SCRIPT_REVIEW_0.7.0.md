## 2. 五つの仮想人格レビュー

### 2.1 日本語会話編集者

旧稿は一文が短すぎ、情報の受け渡しだけに見えた。v5では、言い直し、沈黙、相手への遠慮、茶化し、手を動かしながらの会話を増やした。

合格条件：音読時に、話者名を隠しても主要人物の違いが出ること。

### 2.2 ゲーム・ナラティブデザイナー

旧稿はイベントの意味が「次の目的地説明」に寄っていた。v5では、プレイヤーが戦う理由をパイセンの罪悪感、クマバーソンとTAKUYA、ガンテツとモンキーの関係へ結び直した。

合格条件：各ステージ終了時に、次の目的だけでなく人物関係が一つ変化していること。

### 2.3 ホラー演出家

旧稿の模倣音声中心設定は分かりにくく、恐怖より説明が前に出た。v5では、大学病院からの流出、西新崩壊、死者の再活動という王道へ戻し、音と無音で崩壊を見せる。

合格条件：導入映像で長文説明なしに、普通の夜が壊れたことが理解できること。

### 2.4 群像劇脚本家

旧稿は全員が順番に一行ずつ喋る構造だった。v5では、全員を毎場面へ参加させず、場面ごとに主役と関係性を絞った。

合格条件：クマバーソン、ババヤガ、センセイ、ノイズが、それぞれ別の理由で同じ行動へ参加しているように見えること。

### 2.5 サウンドディレクター

旧稿はBGMやSEが後付けだった。v5では、BGM停止、無音、環境音、ボイスダッキング、警告音を台本内で指定した。

合格条件：実音源実装後、台詞が戦闘SEに潰れず、無音が演出として機能すること。
