# 西新世紀末物語 — 0.7.0脚本 五人格レビュー

更新日：2026-07-17  
対象：`SCENARIO_0.7.0_COMPLETE.md` / `outbreak-origin-v3`

## 判定方法

各レビュー人格は、初稿をそのまま肯定せず、次の観点で不合格箇所を指摘し、修正後に再判定した。

## 1. 日本語会話編集者

### 初稿で止めた点
- 役割説明をそのまま喋る台詞。
- 全員が同じ速度で正しい判断をする場面。
- パイセンの語尾だけで人物を作る箇所。
- 質問に毎回答え、沈黙や言い直しがない会話。

### 修正
- 質問を無視する、答えを遅らせる、相手の行動を先に止める反応を追加。
- クマバーソンは食事、センセイは患者、ババヤガは損失と帰宅、ガンテツは人数という異なる判断基準で話すよう整理。
- 一文を短くし、音読時に息継ぎできない長台詞をト書きまたは複数の応答へ分解。

### 最終判定
条件付き合格。声優収録前または実装後に、全台詞の読み合わせをもう一度行う。

## 2. ゲーム・ナラティブデザイナー

### 初稿で止めた点
- 会話だけで事件が進み、プレイヤーの戦闘目的と結びつかない。
- 加入人物が結果画面で突然増える。
- 長い導入説明の後にようやく操作が始まる。

### 修正
- 導入映像後、CRAWLERの日常、救難信号、編成へ接続。
- Stage 1は救助と感染拠点、Stage 2は避難車両、Stage 3はTAKUYAと研究記録、Stage 4〜6は地下輸送線封鎖へ連鎖。
- ガンテツとモンキーは戦闘中に役割を見せた後、関係と加入理由を会話で成立させる。
- 戦闘中通信は最大三往復へ制限。

### 最終判定
合格。実装時に会話発火条件とステージ状態を一致させること。

## 3. ホラー演出家

### 初稿で止めた点
- 奇妙な模倣音声を中心へ置き、ウイルス流出と都市崩壊が見えない。
- 怪物の怖さを台詞で説明する。
- 常時BGMで感情を指定する。

### 修正
- 架空大学病院の電源断、隔離扉、感染動物、救急搬入口、西新の夜という因果を映像化。
- 犬が倒れてから立ち上がる場面、店の防火扉の向こうの電話、停電後の無音で恐怖を作る。
- TAKUYAは研究被験者とクマバーソンの知人という二重の意味を持たせる。
- 無音、環境音、物語上重要な具体音を台本へ固定。

### 最終判定
合格。実装で暗さだけに依存せず、音源位置と見えない危険を確認すること。

## 4. 群像劇脚本家

### 初稿で止めた点
- 全員が同じ目的で動き、反対や迷いがない。
- 人物同士に過去がなく、戦闘後に関係が変わらない。
- 冗談が危機を無効化する。

### 修正
- パイセンは後輩を残した罪悪感、クマバーソンはTAKUYA、ガンテツとモンキーは十一年の関係を持つ。
- 救助、証拠保全、撤退、燃料、怪我人を巡って判断を衝突させる。
- ブラックユーモアは恐怖から逃げるための人物反応として限定。
- 章末でパイセンが「今いる人数を数える」行動へ変化する。

### 最終判定
条件付き合格。差し替えキャラクターの正式アート確定後、外見と台詞の不一致を再監査する。

## 5. サウンドディレクター

### 初稿で止めた点
- 「怖いBGM」など抽象指示だけで、開始・停止・復帰がない。
- ボイス、銃声、チェーンソー、環境音の優先順位がない。
- スマートフォンでの聴感条件がない。

### 修正
- 導入の各場面に環境音、SE、BGM、無音を指定。
- 会話時の3〜5dBダッキングと復帰時間を指定。
- 箸、電話、隔離扉、エンジン、盾、工具など、物語を伝える具体音を優先。
- 戦闘時の同時発音、クリップ防止、AudioContext復帰、スマートフォン本体スピーカー確認を受入条件へ追加。

### 最終判定
合格。実音源が仮素材なら、脚本合格とは別にサウンド品質未達として停止する。
