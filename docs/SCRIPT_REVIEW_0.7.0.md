## 2. 五つの仮想人格レビュー

### 2.1 日本語会話編集者

v5はまだ、台詞が「正しい会話」止まりで、漫画・ゲーム内で読みたくなる跳ね方が弱かった。v6では、研究員の緊迫と隠蔽、パイセンとクマバーソンの軽口、センセイの切り返し、ババヤガの数字ネタ、ガンテツとモンキーの腐れ縁を強めた。

合格条件：音読時に、説明ではなく人間同士の反応として聞こえること。

### 2.2 ゲーム・ナラティブデザイナー

v5は物語の筋は通ったが、プレイヤーが最初に浴びる画として弱い。v6では、死後反応のラット、ケージ、排水、紙コップ、首輪、唐揚げ、弁当、盾の名前など、操作前から記憶に残る小道具を増やした。

合格条件：各ステージで「その場面だけの絵」と「次に進む理由」が同時に残ること。

### 2.3 ホラー演出家

研究員のやり取りを、圧や専門語の確認ではなく、死んだはずの実験動物が全員こちらを見ている異常へ変更した。恐怖は説明ではなく、ラット、首輪、無音、咬傷を見せない切り方で出す。

合格条件：導入の研究棟から西新崩壊まで、長文説明なしで事故のヤバさが伝わること。

### 2.4 群像劇脚本家

v6では、全員を均等に喋らせず、場面ごとに軸を絞る。Stage 1はパイセンと救助、Stage 2は白衣の男と人命優先、Stage 3はクマバーソンとTAKUYA、地下編はガンテツとモンキーの関係を中心にする。

合格条件：台詞を削っても人物の関係が見えること。

### 2.5 サウンドディレクター

v6では、漫画的なコマ割りに合わせて、音の切断、紙コップ、首輪、ケージ、クラクション、無音を強調した。音の密度を上げた後に切る構造で、台詞だけに頼らない。

合格条件：実音源実装後、笑い・恐怖・沈黙のタイミングが潰れないこと。
