# 西新世紀末物語 — 0.7.0脚本 五人格レビュー

更新日：2026-07-17  
対象：`SCENARIO_0.7.0_COMPLETE.md` / `outbreak-origin-v5`

## 判定方法

各レビュー人格は、初稿をそのまま肯定せず、次の観点で不合格箇所を指摘し、修正後に再判定した。

## 1. 日本語会話編集者

### v4で止めた点
- 台詞が短すぎて、全員が状況確認用の端末のように話している。
- 情報が正確すぎ、言いよどみ、反論、茶化し、沈黙が不足している。
- パイセンの口調はあるが、痛みや照れを隠す会話が薄い。
- クマバーソンとババヤガ以外の人物が機能名に近い。

### v5修正
- 黙る、飲み込む、冗談で逃げる、行動で返す会話を追加。
- パイセンのユウタへの罪悪感を、弁当箱、救助判断、全員で帰る台詞へ接続。
- クマバーソンは食事で人間を守る人物として一貫させ、危機時ほど声を落とす。
- センセイは説明係ではなく、撤退や命の優先順位を突きつける人物へ調整。
- ガンテツとモンキーに、自己犠牲とそれを嫌う腐れ縁を付与。

### 判定
条件付き合格。実装後に音読監査を必須とする。

## 2. ゲーム・ナラティブデザイナー

### v4で止めた点
- 物語の筋は通ったが、プレイヤーの操作へ移る前の感情の引きが弱い。
- 救助、証拠、加入、次ステージへの動機が説明寄り。
- 章末の余韻がシステム報告に寄りすぎる。

### v5修正
- 導入で日常会話、店の崩壊、ユウタを残す判断を明確化。
- 43日後の弁当箱で、過去と現在の救助動機をつなげた。
- Stage 1は救助対象の子どもにユウタの弁当を渡すことで、パイセンの変化を見せる。
- Stage 2は白衣の男と避難民の対比で、研究資料より人命を優先するドラマへ整理。
- Stage 3はTAKUYAの番号化を拒むクマバーソンの感情を中心にした。
- 西新駅編はガンテツとモンキーの関係が、加入理由とゲーム役割へつながるよう修正。

### 判定
合格。実装時に会話発火条件とステージ状態を一致させること。

## 3. ホラー演出家

### v4で止めた点
- 感染や恐怖の説明がまだ台詞に寄っている。
- 怖さより設定の整理が前に出る箇所がある。
- 音と無音の効果が人物の反応と連動しきっていない。

### v5修正
- 病院地下では研究員同士の言い訳と震えから事故を見せる。
- 救急搬入口では犬の咬傷を見せすぎず、起き上がる瞬間で恐怖を作る。
- 西新崩壊では店内の防火扉、電話、停電、クラクションを使い、見えない恐怖を強化。
- TAKUYA戦では怪物そのものより、クマバーソンが名前を捨てられない痛みを恐怖の中心にした。
- 改札喰いは巨大咆哮ではなく、金属と肉の擦れる音、封鎖扉の時間制限で圧迫感を作る。

### 判定
条件付き合格。アートと音響が安い場合は脚本も死ぬため、実音源・実画面で再監査する。

## 4. 群像劇脚本家

### v4で止めた点
- 各人物の役割は立ったが、会話内の衝突と気まずさが足りない。
- 仲間が仲間になっていく手触りが弱い。
- 笑いが単発の小ネタに寄っている。

### v5修正
- パイセンとセンセイ、クマバーソンとパイセン、ガンテツとモンキーに衝突と遠慮を追加。
- ババヤガの笑いは帰宅願望、数字、妻への恐怖へ限定し、場を壊さない範囲に制御。
- クマバーソンの「数え間違い」を、ユウタの弁当、救助者の食事、エピローグまで通す。
- ガンテツが自分を人数に入れない癖を、パイセンとセンセイが止める流れにした。
- モンキーは設備だけでなく、ガンテツを置いていかない人物として描く。

### 判定
合格。正式アート確定後、見た目と台詞の不一致を再監査する。

## 5. サウンドディレクター

### v4で止めた点
- サウンド指定はあるが、会話の感情と結びつかない箇所がある。
- 物語上重要な具体音が不足している。

### v5修正
- ユウタのスマホ、箸が落ちる音、盾に名前を書く／消す行為、工具、台車、封鎖扉など、感情と結びつく具体音を増やした。
- 導入直前とタイトル前の無音を維持。
- TAKUYAの写真を見ている間はBGMの音数を減らす。
- エピローグは食器音と人の声を主役にし、BGMを薄くする。

### 判定
条件付き合格。仮音源での完成扱いは禁止。
