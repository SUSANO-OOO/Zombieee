# 1. あらすじ

## 1.1 プレイヤー向けあらすじ

感染拡大によって孤立した福岡市西新・早良区。移動拠点で生存者を探すパイセンたちは、早良区役所から途切れ途切れの救難信号を受信する。

区役所へ続く道は、西新商店街を覆う巨大な感染拠点に塞がれていた。一行は商店街で取り残された生存者を救い、現地で戦い続けていたクレイジーキングと大庭豪に出会う。

早良区役所では、真壁玲奈、クマバーソン、ババヤガが避難民の撤収を続けていた。一行は救援車両を守り抜くが、区役所から送られた救難音声と同じ声が、無人の西新防衛線からも流れていることを知る。

声に誘われる次の犠牲者を止めるため、一行は防衛線へ向かう。そこで待っていたのは、人間の声をまねる感染体の群れと、巨大変異個体TAKUYAだった。

TAKUYAと感染拠点を破壊した一行は、西新・早良区間の主要避難経路を確保する。しかし直後、百道浜方面から新たな救難信号が届く。

序章は、パイセンたちがその声へ応答し、次の区域へ向かうところで終わる。

## 1.2 制作・実装向け詳細あらすじ

序章の中心は、「偶然同じ場所に生き残った人間たちが、救助を重ねるうちに一つの部隊になっていくこと」である。

物語開始時のパイセンたちは、まだ軍隊でも英雄でもない。移動拠点を維持し、無線を拾い、助けられる範囲の人間を助けているだけである。早良区役所から届いた最初の救難信号は生きた人間の送信だが、その後に同じ音声が不自然に反復する。

区役所へ向かう唯一の現実的な経路は西新商店街である。商店街では、薬局屋上に大人二人と子ども一人が取り残されている。クレイジーキングは罠とチェーンソーで感染体を食い止め、大庭豪は防火扉を押さえて屋上への退路を守っている。一行は感染拠点を破壊し、三人を救出する。

早良区役所では、真壁玲奈が避難車両の撤収を指揮し、クマバーソンが負傷者搬送と食料管理を担い、ババヤガが庁舎上階から特殊個体を排除している。撤収中、瓦礫の下に残された負傷者を救うか、車両の発進を優先するかで現場判断が衝突する。一行は両方を成立させ、最後の車両を送り出す。

撤収後、同じ救難音声が西新防衛線から流れていると判明する。原因は説明し切らない。感染体が音声をまねているのか、感染拠点が通信機器を利用しているのかも、この時点では断定しない。ただし、放置すれば救難信号を信じた生存者が防衛線へ誘い込まれる。

西新防衛線では、街側へ向けられた障害物と放棄された装備が、過去に内側から感染者を封じ込めようとした痕跡を示す。真壁は退却判断が遅れたことを認めるが、説明や謝罪に時間を使わず、現在流れている声を止めるため前へ進む。

巨大変異個体TAKUYAは、人間の声を攻撃や誘導に利用する。TAKUYA撃破後も感染拠点は活動を続け、一行は奥へ進んで破壊する。これにより西新・早良区間の主要避難経路が開通する。

最後に百道浜方面から、反復ではない新しい救難信号が届く。パイセンは初めて自分から無線へ応答し、一行は百道浜へ向かう。

## 1.3 物語の到達点

- プレイヤーが、区役所救助のため商店街を突破する目的を理解できる
- 各ステージの戦闘目的が物語上の目的と一致する
- 仲間の加入が勧誘台詞だけで決まらず、本人の事情と行動から成立する
- パイセンが、反射的に飛び出す若者から、他人へ応答する側へ一歩進む
- 救難信号を信じる緊張と、偽の声に利用される不気味さを残す
- TAKUYA撃破と感染拠点破壊に明確な達成感を持たせる
- 百道浜編へ自然に接続する

## 1.4 序章で確定しないこと

- 感染発生の原因
- 感染体が声をまねる仕組み
- TAKUYAという名称の由来
- TAKUYAが固有個体か、同系統の識別名か
- 百道浜の救難信号が完全に安全かどうか

# 2. 会話設計

## 2.1 共通ルール

- 台詞は「情報を説明するため」ではなく、「相手へ働きかけるため」に発生させる。
- 質問には質問の答えを返す。格好いい比喩へ逃げない。
- 同じ情報を複数人が言い換えて繰り返さない。
- 一人の発言を別の人物が遮る、聞き返す、否定する、言い直す場面を入れる。
- 全員を常に冷静で有能な状態にしない。
- 怖さ、迷い、苛立ち、安堵を短い反応で見せる。
- 人物紹介は行動を先に見せ、名前は会話の必要が生じてから出す。
- 戦闘中は一発言を短くし、画面上の変化と同じ内容を長く説明しない。
- ブラックユーモアは緊張を緩めるために使い、死や救助の直後へ機械的に置かない。
- クマバーソンの弁当、ババヤガの妻と意味不明発言は、人物を判別できる程度に限定する。
- クレイジーキングは奇人として説明せず、罠、燃料管理、危険な接近戦で異常性と実力を見せる。
- パイセンの固定台詞は重要場面で一度ずつ使用する。
- 「〜だ」「〜する」「了解」を全員へ均等に配らない。
- 日本人同士の会話として、短い相づち、沈黙、言い直しを許可する。

## 2.2 人物ごとの会話目的

### パイセン

怖さを隠し切れない24歳の学生。専門家ではない。人を置いて逃げる判断ができず、先に身体が動く。

- 普通の若い男性の日本語
- 「〜っす」は自然な頻度に限定
- 難しい作戦用語や長い号令を言わせない
- 怒ると語彙が減る
- 仲間の言葉を受けてから判断する

固定台詞：

- 「もう、いいよ！ こいよ！」
- 「さすがに許せないっす」
- 「あああああああうああ！」

### 水城奈々

通信・地図・情報分析担当。正確だが、人間の命を数値として扱わない。断定できない時は断定しない。

### 黒木凛

射撃手。観察した事実を短く伝える。危険な判断にはすぐ反対する。感情を隠すが、仲間が戻るまで射線を切らない。

### 橘迅

遊撃手。先行して道を作る。軽口は緊張を隠すためであり、常に上手いことを言う人物ではない。

### 白石直人

衛生兵。助かる人間を優先する現実主義者。冷たく見える判断もするが、見捨てることへ慣れてはいない。

### 大庭豪

破砕兵。必要なことだけを言う。怪力を誇示せず、一人で無理な時は手を借りる。

### 真壁玲奈

区役所撤収の現場責任者。判断力はあるが、過去の遅れを抱えている。自分を正当化せず、現在の仕事へ戻る。

### クレイジーキング

罠、敵の寄り方、燃料を観察して戦う。高いテンションだけで喋らない。普通の質問には普通に答えられる。

### クマバーソン

弁当屋。自然な博多弁。食事の話は生命維持と地域への愛着につながっている。深刻な場面ではふざけない。

### ババヤガ

27歳の証券会社員兼ヒットマン。冷静な標準語が基本。時々、文法は通るが意味の分からない表現を真顔で使う。本気になると冗談を完全に止める。

# 7. 実装上の禁止事項

- Codexによる台詞の意図を変える要約・言い換え
- 旧名、過去名、登録名を使った人物証明
- 現在の人物名を過去記録へ登場させる展開
- TAKUYAの名称由来を独自に追加すること
- 全員を軍人口調へ統一すること
- 全員へ決め台詞を均等配布すること
- 会話を一人ずつの状況説明へ戻すこと
- クマバーソンの博多弁を語尾だけで作ること
- ババヤガを寡黙な謎の殺し屋へ変更すること
- クレイジーキングを格言だけ話す狂人へ変更すること
- 戦闘スプライトを会話ポートレートへ流用すること
- 不明な無線へ水城奈々の立ち絵を流用すること
- あらすじと異なるステージ目的へ変更すること
- 重要な救助・死亡・勝利の直後に必ず冗談を入れること
