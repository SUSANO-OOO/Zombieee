## 1. 使用規則

- この文書の話者、セリフ、順序を0.7.0台本の正本とする。
- Codexは要約、言い換え、台詞追加、語尾統一を行わない。
- 表示幅による改行だけをUI側で行う。
- 戦闘中通信は短い非遮蔽表示を基本とし、操作を止めない。

- 同じ人物でも状況に応じて口調を変える。口癖を毎行へ付けない。

## 2. 全体あらすじ

### 短文

早良区役所から続いていた不自然な救難信号。その下に、録音ではない呼吸が混じった。パイセンたちは感染体が地上へ流れ出す西新駅へ降り、閉じ込められた生存者と、駅そのものを巣に変えつつある感染源へ向かう。

### 長文

商店街、早良区役所、西新防衛線を突破した生存者部隊は、救難信号を模倣する感染群とTAKUYAを退けた。しかし通信担当ノイズは、百道浜方面から届く反復音声の底に、別の送信を発見する。声は西新駅の地下から発せられ、駅構内では警備員ガンテツが生存者を守り、保線作業員モンキーが線路側で設備を動かし続けていた。感染体は駅の放送、配線、改札、保守設備へ広がり、人間の声を使って地上から新たな獲物を呼び込もうとしている。部隊は生存者を救い、感染の流出経路を封鎖し、崩れ始めた地下から戻らなければならない。
