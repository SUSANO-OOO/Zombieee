# 西新世紀末物語 — 0.7.0完全台本

更新日：2026-07-17  
script version：`chapter-nishijin-station-v2`

## 1. この台本の核

市内感染拡大から四十三日。生存者を乗せて移動する大型拠点は、西新周辺で途切れた救難無線を拾う。

感染体の一部は、人間の声を覚え、電波や放送設備を使って繰り返す。ただし、言葉の意味までは理解していない。

この章で描くのは「偽物の声を見破る話」ではない。怖くても、罠かもしれなくても、まだ本物かもしれない声を見捨てない人間たちの話である。

## 2. タイトル画面用あらすじ

人の声を真似る感染体が、西新の無線を乗っ取った。

移動拠点で生き延びるパイセンたちは、その雑音の底に残った本物の救難信号を追い、閉鎖された街へ入る。

## 3. 会話の原則

- 人物はプレイヤーへの説明書として喋らない。
- 作戦情報は、必要な人物が必要な瞬間にだけ伝える。
- 全員を毎場面へ参加させない。沈黙、視線、行動も人物表現として扱う。
- 口癖は特徴であって文章生成規則ではない。
- パイセンは勢いだけの人物にしない。怖さを認めたうえで身体を動かす。
- ノイズは万能解説役にしない。聞き分けを誤る恐怖と責任を持たせる。
- クマバーソン、ババヤガ、クレイジーキングの笑いは緊張を壊すためではなく、人間が壊れないためのものとして使う。
- 戦闘中通信は一往復から三往復まで。操作を止めない。
- Codexは台詞を要約、言い換え、追加しない。演出上必要な改行だけを行う。

## 4. 実装上の必須確認

- `水城奈々`、`橘迅`、`黒木凛`、`白石直人`、`大庭豪`、`真壁玲奈`を0.7.0表示へ残さない。
- 旧save IDはmigrationへだけ残し、プレイヤー向けには新人物を表示する。
- `西神`表記を一件も残さず、すべて`西新`へ統一する。
- タイトル画面のあらすじは本稿の二段落だけを使い、長文を常時表示しない。
- 戦闘前会話は必要な人物に絞り、全員へ一行ずつ割り当てる構造にしない。
- 戦闘中通信は最大三往復を原則とし、操作を止めない。
- 感染体の模倣音声は、同一章内ですでに発話された台詞または救難録音を素材にする。都合のよい新情報を喋らせない。
- パイセンの`〜っす`、クマバーソンの博多弁、ババヤガの奇妙な言葉を毎行へ機械的に付けない。
- ノイズは説明装置ではなく、聞き分けを誤る可能性と責任を持つ人物として扱う。
- ガンテツの丁寧語は一場面に一度程度の笑いを生んでもよいが、単なる日本語間違い要員にしない。
- モンキーは機械だけを愛する人物にせず、設備を守ることが人間を守る行為になっていると伝える。
- クマバーソンとTAKUYAの関係を削除しない。TAKUYA戦の感情的な中心とする。
- 改札喰いが味方の声を再生する場面では、音だけを真似る感染体と、言葉の意味を引き受ける人間の対比を維持する。
- 章終了後の規則的な打音は次章の可能性だけを示し、発信者や感染原因を確定しない。
