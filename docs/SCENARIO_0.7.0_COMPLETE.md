## v6の漫画寄り会話方針

- 研究事故、避難、戦闘前会話は、漫画のネームのように「絵で分かる状況」「一発で刺さる台詞」「短いツッコミ」「無音のコマ」を組み合わせる。
- 研究員パートは専門用語の読み上げではなく、死んだはずの実験動物が立つ、管理者が隠蔽を選ぶ、若手がそれに気づく、という絵で理解できる恐怖にする。
- ユーモアは説明の息抜きではなく、人間が怖さを誤魔化すために出るものにする。滑った冗談は別人物が拾うか切る。
- キャラの会話は「役割」ではなく「その人が今、何を守りたいか」から出す。
- パイセンは勢いで喋り、痛いところを突かれると急に黙る。
- クマバーソンは飯の話で人を現実へ戻す。ふざけているようで、誰より人数を見ている。
- ババヤガは数字と妻で現実に戻す。意味不明語は乱発せず、ここぞで変な比喩を出す。
- センセイは正論機械にしない。怒る時ほど短く、疲れている時ほど優しい。
- ノイズは説明装置にしない。必要な情報を絞り、判断を他人へ押しつけない。
- ガンテツとモンキーは、ツッコミ合いの裏に「相手が死にそうな癖を知っている」関係を置く。
- 戦闘中通信は原則最大三往復で、操作を止めない。
- BGM、環境音、SE、無音、ボイスダッキングは台本の一部として実装する。

## 実装規則

- fresh saveで「物語を始める」を押した直後、タイトル画面の長文ではなく導入映像を再生する。
- 初回導入はスキップと再視聴に対応する。
- スキップ時は二行の短い要約だけを表示し、長文あらすじを常時読ませない。
- Codexは台詞、場面順、人物関係、物語設定を要約、言い換え、追加、削除しない。
- BGM、環境音、SE、無音、ダッキングは台本の一部として実装する。
- 本作の大学病院は架空であり、実在の大学・医療機関とは無関係。
- 正式採用は実ブラウザ、音読、実音源、スマートフォンでの受入後とし、文書作成だけで完成扱いにしない。
