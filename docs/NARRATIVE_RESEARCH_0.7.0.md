## 2. 現在の固定事項

- 店舗は大衆居酒屋「くまや」
- 主人公は発生前の常連客
- 発生夜の避難とCRAWLER確保を経て指揮官となる
- プレイヤーは無言、名前入力なし、会話分岐なし
- キャラクターボイスなし
- BGM、環境音、SE、必要な無音だけを使用
- アート制作から実装・QAまでCodexが担当
- パイセン、クマバーソン、ババヤガ、クレイジーキングの外見は保全
- その他7名とノイズは全面再設計
- `ロッカ`は不採用、正式表示は`レイダー`
- `センセイ`は不採用の仮称。正式名はCodexが決定
- 初期所有はパイセン、ハチ、ミズチ、医療支援、クマバーソン、ババヤガ
- クレイジーキング、ガンテツ、モンキーは物語無料加入
- タタラ、レイダーはキャップ調達
- 新敵は絡手、漏泥、走鬼、改札喰い
- 3レーンを維持し、背景と論理レーンの不一致を禁止
- Safari保存修正を最新mainから取り込み、破損save復旧をschema v5で強化

## 3. 維持する制作原則

- 設定説明より、人物の行動と画面で状況を伝える。
- 台詞は情報の朗読ではなく、人物の価値観と関係から出す。
- 話者名を隠しても人物を区別できる会話にする。
- ユーモアは場面を止めず、恐怖や緊張の隙間で使う。
- 画面、表情、BGM、環境音、SE、無音、次の操作へ役割を分担する。
- 重要場面だけ全画面カットを使い、常時豪華演出へしない。
- 実装後に音読、スマートフォン表示、操作テンポを確認する。

## 4. 不採用の旧原則

- 研究所会話から開始する
- プレイヤー不在の群像劇として進める
- 独自チュートリアル戦闘を追加する
- 人物がゲーム仕様を順番に説明する
- キャラクターボイスやボイスダッキングを完成条件にする
- アート制作をChatGPT／プロデューサー側だけに限定する
