# 西新世紀末物語 — 0.7.0物語・会話設計リサーチ

更新日：2026-07-17

## 1. 調査の範囲と誠実性

本調査は、119作品を比較対象として、公開されている公式紹介、開発者・脚本家インタビュー、プレイ映像・イベント映像、シナリオ解説、既知の作品構造を横断して分析したものである。
119作品を今回の作業時間内に新規で全編プレイ・全巻通読した、という虚偽の主張はしない。
台詞の転載は禁止転載や模倣は行わず、構造・演出・会話設計の原則だけを抽出する。

## 2. 参照した制作論・実務知見

- アニメ脚本は、監督やシリーズ構成の意図を踏まえたプロット、自然な会話、継続的なリテイクを重視する。
- ゲームの会話は文章だけでは完結せず、直前のプレイ、画面、表情、BGM、SE、次の操作との役割分担が必要。
- キャラクターらしい台詞は、甘さや派手さより、人物の価値観・関係・過去と一致していることが先。
- 漫画的な会話は、説明を削った上で、絵、沈黙、短いツッコミ、行動、効果音で意味を補う。
- 台本は音読し、話者名を隠しても誰の台詞か判別できるか、息継ぎや反応が人間の会話として成立するかを確認する。
- サウンドはBGM、環境音、SE、ボイスを場面ごとに設計し、ボイス時のダッキング、無音、音源位置、復帰を含めて実装する。

## 3. v6で採用する漫画寄りの原則

1. 冒頭は設定説明ではなく、異常の絵から始める。
   - 例：死後反応のラットがこちらを見る。
   - 例：首輪の音だけを鳴らして、咬傷そのものは切る。

2. 台詞は情報より反応を優先する。
   - 「何が起きたか」を説明する前に、「見た人がどう誤魔化すか」「どう怒るか」「どう笑って耐えるか」を置く。

3. ユーモアは恐怖の隙間で出す。
   - ただのギャグ挿入にしない。
   - キャラの価値観から出るズレにする。

4. 小道具で感情を持たせる。
   - 唐揚げ、弁当箱、紙コップ、首輪、盾の名前、箸、グラスなど。
   - 説明台詞より記憶に残る絵を優先する。

5. 説明はUIへ逃がす。
   - 経路、条件、チュートリアル、ステージ情報は人物に読ませない。
   - 人物は、自分の判断と感情だけを喋る。

6. 見開き的な山を作る。
   - 発生夜：ユウタを残す。
   - Stage 1：余った弁当を救助した子どもへ渡す。
   - Stage 2：白衣の男がケースではなく担架を持つ。
   - Stage 3：クマバーソンがTAKUYAを番号でなく名前で呼ばせる。
   - 地下：ガンテツが自分を数に入れられる。
   - 終幕：パイセンが今いる人数を数える。
