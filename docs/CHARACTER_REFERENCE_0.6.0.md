## 3. 誤登録の禁止

- 白いワイシャツとサプレッサー付き拳銃の参照画像は`ババヤガ`であり、`ブギーマン`ではない
- `ブギーマン`の人物設定、台詞、内部ID、解放枠を新規作成しない
- 過去に作成した「寡黙で正体不明のブギーマン」案をババヤガへ自動統合しない
- ババヤガは、生活感のある新婚証券社員兼ヒットマンという設定を優先する

## 4. 実装時の画像規則

提供画像には複数ポーズが横並びで収録されている。背景の市松模様は透明表示ではなく画像へ焼き込まれているため、参照コピーをそのまま実行時素材へ使用しない。

実装時は以下を必須とする。

- プロデューサー提供原本をローカル作業環境で照合してから制作する
- 背景を適切に除去し、透過素材として再作成する
- 各ポーズを個別フレームへ正確に分離する
- 隣接フレームの手、腕、武器、血、影、輪郭を混入させない
- 画像比率を崩さず、無理な引き伸ばしや縦長圧縮を行わない
- 戦闘スプライトと会話用ポートレートを別用途として制作する
- 会話用ポートレートは既存キャラクターと同一の画風、光源、輪郭密度へ統一する
- 左右反転で文字、装備、銃、ホルスターの位置が不自然になる場合は反転を使わない
- 全フレームを原寸、拡大、実戦表示サイズで目視確認する
