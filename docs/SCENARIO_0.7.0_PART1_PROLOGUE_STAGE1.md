## 1. 実装規則

- fresh saveで「物語を始める」を押した直後、タイトル画面上の長文ではなく導入映像を再生する。
- 導入映像は初回のみ自動再生。設定・回想から再視聴できる。
- スキップ可能。ただし初回スキップ時は二行の短い要約を表示する。
- Codexは台詞の要約、言い換え、追加、語尾統一を行わない。
- ト書き、BGM、環境音、SE、無音、ダッキングを台本の一部として扱う。
- 戦闘中通信は一往復から三往復。戦闘操作を止めない。
- 全員へ均等に一行ずつ喋らせない。
- 地名表記はすべて`西新`へ統一する。
- 本作の大学病院は架空であり、実在の大学・医療機関とは無関係。
