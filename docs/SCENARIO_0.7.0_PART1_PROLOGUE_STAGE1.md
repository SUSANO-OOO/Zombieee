# 西新世紀末物語 — 0.7.0完全台本 PART 1

更新日：2026-07-17  
script version：`outbreak-origin-v8`

## 1. 実装契約

- 主人公はプレイヤー。発生前は大衆居酒屋「くまや」の常連客。発生から四十三日後はCRAWLERの指揮官。
- プレイヤーは基本的に無言。固定台詞、性格選択、会話分岐を実装しない。
- プレイヤー名入力なし。UI上は`指揮官`で成立し、台詞へ変数名を挿入しない。
- プロローグに独自戦闘、操作チュートリアル、追加ミッションを置かない。
- 最初の戦闘は既存の`Stage 1 西新商店街`。
- 物語イベントの登場人物と実際の7枠編成は分離してよい。戦闘中barkは出撃中の場合だけ表示する。
- プロローグは初回自動再生、スキップ、回想再視聴に対応する。
- ストーリー会話の読み上げボイスは使用しない。既存の戦闘中キャラクターボイスは維持し、BGM、環境音、SE、無音、画面転換を台本の一部として扱う。
