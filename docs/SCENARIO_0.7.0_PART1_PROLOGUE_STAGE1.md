## 1. このPartの狙い

導入は説明ではなく、漫画の冒頭のように「普通の夜が壊れる絵」と「短い会話」で見せる。研究棟では何が起きたかを専門用語で読ませず、死んだはずの実験動物が立つ、隠蔽を選ぶ大人、逃げ遅れる若手、という絵で理解させる。

- 長文あらすじをタイトルに置かない。
- 「Tウイルス」は最低限だけ見せ、詳細はStage 2以降で開示する。
- パイセンの罪悪感は説明しすぎず、ユウタを残した場面、四十三日後の数え癖、救助時の反応で見せる。
- 会話は全員へ均等に台詞を振らない。黙る、遮る、言い直す、茶化す、手を動かす。
- 研究員パートは「圧が落ちてます」では弱いので、実験動物の異常をコマで見せる。
- BGM、環境音、SE、無音、ボイスダッキングは台本の一部。
