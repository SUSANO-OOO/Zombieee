# 西新世紀末物語 — 物語正本 0.7.0

更新日：2026-07-17  
物語版：`outbreak-origin-v1`

## 1. 作品の約束

これは、福岡市・西新を舞台にした王道の都市型ゾンビサバイバルである。

秘密研究、流出事故、一夜で崩壊する繁華街、死者の再活動、逃げ遅れた人々、仲間を集めて前へ進む移動拠点。
奇をてらった設定を先に出さず、誰が見ても状況と目的を理解できる物語から始める。
独自性は、西新・早良区の生活、人物の距離感、ブラックユーモア、移動拠点を使ったリアルタイム戦闘、群像劇から作る。

## 2. 世界設定

### 2.1 架空の研究機関

`福岡市西部医科大学附属病院`は、本作のために作った架空の大学病院であり、実在の大学・病院とは関係がない。

同病院の地下研究棟では、重度脳損傷と心停止後の神経再生を目的とする極秘研究`T計画`が進められていた。
研究用ウイルスの現場呼称は`Tウイルス`。

### 2.2 Tの意味

表向きの資料では`T`は組織再生を示す記号として扱われる。
しかし実際には、最初に長時間の蘇生反応を示した被験者`TAKUYA`の名から研究員が呼び始めた符号である。

この事実はゲーム開始時には明かさない。
Stage 2で`T-01`という記録が見つかり、Stage 3でTAKUYA本人とT計画の関係が判明する。

### 2.3 流出

発生日時：物語開始の四十三日前、金曜日の夜。

地下研究棟で原因不明の電源断と隔離扉の誤作動が同時に発生。
感染した実験用ラットが排水・搬入口へ逃げ、研究棟に収容されていた犬と猫にも感染が広がる。
咬傷、血液、体液を通じて救急搬送者、警備員、病院職員へ拡大。
感染者は高熱、錯乱、心停止を経た後、運動機能だけが再活動し、生者へ襲いかかる。

病院側は事故を隠そうとして初動を遅らせる。
感染した動物と救急車両、人の移動によって早良区へ流入し、夜間人口が集中していた西新の繁華街で爆発的に拡大。
西新は一夜で崩壊する。

### 2.4 現在

発生から四十三日。
行政機能と通信網は寸断され、生存者は大型移動拠点`CRAWLER`を住居、輸送、作戦基地として使用している。
プレイヤーはCRAWLERから部隊を編成し、救助、物資回収、感染源破壊、経路封鎖を行う。

## 3. 主人公の核

パイセンは24歳の大学生。
発生の夜、西新の飲み屋でアルバイトをしていた。
混乱の中で複数の客を逃がしたが、同じ大学の後輩一人を店内に残したまま退避した。
本人は合理的な判断だったと理解しているが、「もう一度戻れば助けられたかもしれない」という罪悪感を抱えている。

そのため、罠かもしれない救難信号にも過剰に反応する。
ただの熱血漢ではない。見捨てた記憶が、怖くても先に動く理由になっている。

## 4. 主要人物のドラマ上の役割

- パイセン：見捨てた罪悪感と、誰も置いていかないという無理な誓い。
- クマバーソン：食事と生活を守ることで、人間らしさを維持する。TAKUYAを災害前から知る。
- ババヤガ：生きて妻の元へ帰ることを最優先する現実主義者。冷静だが仲間を数字だけでは扱えない。
- クレイジーキング：壊れて見えるが、現場の危険を最も正確に読む。救助対象を守るため一人で残っている。
- ノイズ：通信担当。万能解説者ではなく、誤情報を出せば人が死ぬ責任を背負う。
- センセイ：救える人数を増やすため、撤退を選べる人物。パイセンの無茶に真正面から反対する。
- ハチ：経路を作る。単独行動が得意だが、戻る約束を守ることに執着する。
- ミズチ：必要な敵だけを落とす。感情を隠すが、仲間の呼吸や動きをよく見ている。
- タタラ：壊すべき場所を見極める。暴力ではなく構造を扱う。
- ロッカ：戦場のリズムを整える。会話でも他人のテンポを崩さない。
- ガンテツ：誰かを守るため最後に残ろうとする癖がある。
- モンキー：設備を守ることが人を守ることだと知っている工兵。

## 5. 0.7.0までの全体構成

### プロローグ：西新が死んだ夜
大学病院地下でTウイルスが流出。
感染動物と救急搬送を通じて早良区へ拡大。
パイセンが働く西新の飲み屋街で最初の大規模発生。
逃走、停電、火災、地下鉄封鎖。
タイトル表示。

### 第一幕：四十三日後
CRAWLERと生存者部隊の現在を短い生活場面から見せる。
西新商店街から救難信号。
危険と物資不足を理由に反対する者、戻ると決めるパイセン。
プレイヤーが初めて編成する。

### Stage 1：西新商店街
救助と感染拠点破壊。
クレイジーキングとタタラに遭遇。
病院搬送箱とT計画の刻印を発見。

### Stage 2：早良区役所
避難車両を守る。
病院研究員の端末から、`T-01 移送失敗`と地下輸送経路の存在を知る。
クマバーソンが記録写真のTAKUYAを見て、知人だと気づく。

### Stage 3：西新防衛線・TAKUYA
TAKUYAと交戦。
クマバーソンは知人を救える可能性に迷い、パイセンは止める役を担う。
戦闘後、TAKUYAがT計画最初の被験者で、ウイルス呼称の由来だったと判明。
地下鉄の保守線へ研究資料が運ばれていた証拠を得る。

### Stage 4：西新駅・改札区域
地下へ向かい、ガンテツと生存者を救う。
感染中継点を破壊。
病院の輸送箱が改札奥へ運ばれていたと判明。

### Stage 5：西新駅・ホーム／線路区域
モンキーと合流。
保守台車で生存者と物資を護送。
新種感染体と戦いながら、地下研究輸送線の記録を回収。

### Stage 6：西新駅・保守トンネル／封鎖区域
地上へ続く感染流出路を封鎖。
大型特殊個体`改札喰い`を撃破。
全員で帰還する。
大学病院地下の一部電源が現在も稼働していることが分かり、次章の目的が生まれる。

## 6. 情報開示

ゲーム開始時に分かる：
- 大学病院で事故が起きた。
- 感染者は死亡後に再活動する。
- 西新が一夜で崩壊した。
- パイセンは発生の夜を生き延びた。

Stage 1後：
- 病院由来の研究物資が市街地へ運ばれていた。

Stage 2後：
- `T-01`という被験者記録がある。
- 地下輸送経路が存在する。

Stage 3後：
- TAKUYAはT-01。
- Tウイルスの呼称はTAKUYAに由来する。
- 事故は単純な自然災害ではなく、隠蔽と移送失敗で拡大した。

Stage 6後：
- 西新駅ルートは封鎖した。
- 大学病院地下の研究棟は完全停止していない。
- 流出事故が偶然だったかは未確定。

## 7. 禁止事項

- 人の声を真似る感染体を物語の中心設定にしない。
- タイトル画面へ長いあらすじを常時表示しない。
- 実在の大学病院を違法研究機関として描かない。
- 登場人物へ設定資料を朗読させない。
- TAKUYAを理由のない大型敵で終わらせない。
- キャラクターの口癖を毎行へ付けない。
- 全員が同じ正義感で即座に賛成する場面を作らない。
