# 西新世紀末物語 — 物語正本 0.7.0

更新日：2026-07-17  
物語版：`outbreak-origin-v7`

## 1. 作品の約束

これは、福岡市・西新を舞台にした王道の都市型ゾンビサバイバルである。

秘密研究、ウイルス流出、一夜で崩壊する繁華街、死者の再活動、逃げ遅れた人々、仲間を集めて進む移動拠点。

設定の奇抜さを先に見せず、誰が見ても状況と目的を理解できるところから始める。独自性は、西新・早良区の生活感、人物の距離感、ブラックユーモア、CRAWLERから部隊を指揮するゲーム体験で作る。

## 2. 主人公

主人公はプレイヤーである。

- プロローグでは、小料理屋`くまや`に居合わせた生存者の一人。
- 発生から四十三日後は、CRAWLERの部隊編成と作戦決定を担う指揮官。
- 基本的に無言。固定台詞や性格選択で人格を決めない。
- 周囲の人物がプレイヤーへ報告し、視線を向け、指示を待つことで中心性を示す。
- プレイヤー名入力は物語の必須条件にしない。UI上は`指揮官`で成立する。
- 会話選択肢による本筋分岐、好感度、別エンディングは0.7.0で扱わない。

## 3. 世界設定

### 3.1 架空の研究機関

`福岡市西部医科大学附属病院`は、本作のために作った架空の大学病院であり、実在の大学・病院とは関係がない。

同病院の地下研究棟では、重度脳損傷と心停止後の神経再生を目的とする極秘研究`T計画`が進められていた。研究用ウイルスの現場呼称は`Tウイルス`。

### 3.2 Tの意味

表向きの資料では`T`は組織再生を示す記号として扱われる。

実際には、最初に長時間の再活動を示した被験者`TAKUYA`の名から、研究員が使い始めた符号である。

この事実は開始時に明かさない。

- Stage 1：大学病院の搬送箱を発見。
- Stage 2：`T-01`の移送記録とTAKUYAの写真を発見。
- Stage 3：TAKUYA本人とT計画の関係が判明。

### 3.3 流出

発生日時は、物語開始の四十三日前、金曜日の夜。

地下研究棟で隔離事故が発生し、感染した実験用ラットが排水・搬入口へ流出。犬、猫、病院職員、救急搬送者へ感染が拡大した。

感染者は高熱、錯乱、心停止を経た後、運動機能だけが再活動し、生者へ襲いかかる。

病院側は初動で事故を隠し、警告が遅れた。感染した動物、救急車両、人の移動によって早良区へ広がり、金曜夜の西新繁華街で爆発的に拡大した。

研究所内部の会話シーンはゲーム内で直接表示しない。プレイヤーは、くまや周辺の異変、緊急速報、Stage 1〜3で得る証拠から原因へ近づく。

### 3.4 現在

発生から四十三日。

行政機能と通信網は寸断され、生存者は大型移動拠点`CRAWLER`を住居、輸送、作戦基地として使用している。

プレイヤーはCRAWLERの指揮官として、所有ユニットから部隊を編成し、救助、物資回収、感染拠点破壊、経路封鎖を行う。

## 4. 主要人物の物語上の役割

- プレイヤー：無言の指揮官。判断と戦闘操作で物語の中心に立つ。
- パイセン：怖くても先に身体が動く前衛。語録は決め場面に限定する。
- クマバーソン：熱血・パワー型。食事と生活を守ることで人間らしさを維持する。TAKUYAを災害前から知る。
- ババヤガ：基本は冷静な現実主義者。数字と妻への帰宅願望で状況を捉え、崩れると博多弁が出る。
- クレイジーキング：イカれて見えるが、現場の危険判断は正確。漢字とカタカナ中心の短い言葉だけを使う。
- ノイズ：通信担当。指揮官へ必要な情報だけを報告し、判断を代行しない。
- センセイ：負傷者の優先順位と撤退判断を担う。正論を言うためではなく、生存者を増やすために止める。
- ハチ：進入路と帰還路を作る遊撃役。
- ミズチ：特殊個体と弱点を先に見つける射撃役。
- タタラ：構造を見て、壊すべき箇所だけを壊す。
- ロッカ：戦場の音と射撃の間を整える。
- ガンテツ：西新駅の生存者を守る警備員。自分を人数へ入れない癖がある。
- モンキー：設備を守ることが人を守ることだと知る保線作業員。

## 5. 0.7.0の物語構成

### PROLOGUE：西新が終わった夜

崩壊前の小料理屋`くまや`から始める。

プレイヤー、パイセン、クマバーソン、ババヤガの日常を短く見せた直後、路上で感染が始まる。プレイヤーの判断で客の退路が作られ、四人は生き延びる。

戦闘や操作チュートリアルは入れず、短い固定演出で西新崩壊とタイトルへつなぐ。

### CHAPTER 1：発生から四十三日

CRAWLER内の生活と指揮官としてのプレイヤーを示す。

西新商店街から救難信号。プレイヤーが既存の作戦卓・出撃操作でStage 1を選ぶ。

### Stage 1：西新商店街

- 感染拠点を破壊し、薬局二階の生存者五名を救助。
- クレイジーキングと遭遇。
- 大学病院の搬送箱と`T PROJECT`刻印を発見。
- クレイジーキングが加入。

### Stage 2：早良区役所

- 避難車両の始動まで防衛。
- 研究資料より負傷者を優先する指揮官の立場を示す。
- `T-01`の移送記録とTAKUYAの写真を発見。
- クマバーソンがTAKUYAを知人だと認識。

### Stage 3：西新防衛線・TAKUYA

- TAKUYAを制圧。
- クマバーソンは友人として名前を呼び、最後の判断を指揮官へ預ける。
- 病院と西新駅を結ぶ地下輸送経路を発見。

### Stage 4：西新駅・改札区域

- ガンテツと生存者七名を救助。
- 感染中継点を破壊。
- ガンテツの加入条件を満たす。

### Stage 5：西新駅・ホーム／線路区域

- モンキーと合流。
- 保守台車を護衛し、生存者五名と物資を運ぶ。
- 病院から保守トンネルへ続く輸送記録を回収。

### Stage 6：西新駅・保守トンネル／封鎖区域

- 三つの電源を順番に起動。
- 大型特殊個体`改札喰い`と研究容器を封鎖扉の向こうへ押し込む。
- 地上へ続く感染流出路を暫定封鎖。
- 全員で帰還する。

### EPILOGUE

救助者をCRAWLERへ収容。

西新駅側の感染流出は止まるが、大学病院地下の電源が現在も稼働していることが判明し、次期調査地点となる。
