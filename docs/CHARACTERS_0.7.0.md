# 西新世紀末物語 — 0.7.0キャラクター正本

更新日：2026-07-17

## 1. 目的

この文書は0.7.0のプレイアブル11名と非戦闘オペレーター1名について、人物、外見、武器、戦闘役割、固有特性、口調、加入条件を固定する正本である。

Codexは、この文書にない人物設定やセリフを独自に生成しない。技術的に必要なID、数値、アニメーション分割は調整できるが、人物の名前、性格、武器、見た目、関係性を変更しない。

## 2. 共通原則

- プレイアブルは11名。
- 主要クラスは1つ、役割タグは複数。
- 一覧で判別できる固有シルエットを持つ。
- 肖像と戦闘スプライトで顔、髪、体格、服、武器を一致させる。
- 同じ顔、身体、武器を色替えだけで別人物へ流用しない。
- 既存のパイセン素材は保全条件を維持する。
- 固有特性は自動発動または常時効果。個別アクティブスキルは0.8.0へ送る。
- セリフは短く、人間が口にする自然な日本語にする。役割説明をそのまま喋らせない。

## 3. クラスID

- `class-frontline`：前衛
- `class-skirmisher`：遊撃
- `class-marksman`：射撃
- `class-heavy`：重装
- `class-support`：支援
- `class-engineer`：工兵
- `class-berserker`：狂戦

## 4. プレイアブル

### 4.1 パイセン

- `unitId`：`unit-paisen`
- `combatKind`：`brawler`
- 主要クラス：前衛
- 役割タグ：近接、連撃、押し返し、低コスト
- 年齢：24歳
- 立場：学生。災害後に西新の生存者集団へ合流
- 武器：素手
- 外見：現行の人物、体格、服装、素手の構えを非破壊で保全
- 性格：直情的で、怖くても先に身体が動く。敬語と雑な若者言葉が混ざる
- 口調：語尾は必要な場面で`〜っす`。一文ごとに付けない
- 固有特性`ノッてきたっす`：同じ対象へ連続命中すると短時間だけ攻撃速度と押し返しが上がる。対象変更または被弾で段階が落ちる
- 役割：安価な前線維持。強敵を単独で倒すより、敵の足を止めて他の火力へつなぐ
- 代表セリフ：
  - 出撃：`もう、いいよ。こいよ！`
  - 危機：`さすがに、それは通せないっす！`
  - 撤退：`一回下がる！　まだ終わってない！`

- `unitId`：`unit-hachi`
- 旧対応：橘迅／`scout`
- `combatKind`：`scout`
- 主要クラス：遊撃
- 役割タグ：高機動、近接、レーン移動、対高速
- 年齢：23歳
- 以前の仕事：自転車便、夜間配送、非公式レースの案内役
- 武器：短く切ったバール、折り畳み式の打撃棒
- 外見：小柄で筋肉質。黒髪の刈り上げ、擦れた黄色いメッセンジャージャケット、膝当て、首に自転車用ゴーグル。大きなバッグは持たず、腰へ工具をまとめる
- 性格：考えるより先に走るように見えるが、道、距離、出口を常に数えている。無謀ではなく、逃げ道を作るために危険へ入る
- 口調：短文。相手を急かすが、弱っている人間には声を落とす
- 固有特性`抜け道`：担当レーンに高速敵または後方侵入敵が現れたとき、最初の接敵まで移動速度が上がる。最初の一撃は短い足止めを与える
- 役割：走鬼、runner、後方へ抜けた敵の回収
- 代表セリフ：
  - 出撃：`道、見えた。先に行く`

  - 撤退：`戻る。置いてはいかない`

## 5. 非戦闘オペレーター

### ノイズ

- `characterId`：`guide-noise`
- 旧対応：水城奈々／`guide-mizuki-nana`
- 戦闘参加：しない
- 年齢：31歳
- 以前の仕事：コミュニティFMとライブハウスの音響技術者。災害時に臨時無線局を運用
- 役割：通信、地図、音声分析、作戦情報、記録

  - `同じ声が二つあります。一つは録音。もう一つは、今、息をした`
  - `感想は帰ってから。右の通路が閉じます`
