# 西新世紀末物語 — 0.7.0キャラクター正本

更新日：2026-07-17

## 1. 目的

この文書は0.7.0のプレイアブル11名と非戦闘オペレーター1名について、内部ID、人物像、武器、戦闘役割、固有特性、口調、加入条件を固定する。

主人公はプレイヤーであり、この11名のいずれかではない。物語上のプレイヤーの役割は`STORY_BIBLE_0.7.0.md`を正本とする。

Codexは、人物名、性格、武器、関係性、台詞の口調を独自に変更しない。表示名が今後変更されても、内部IDとsave migrationは維持する。

## 2. 共通原則

- プレイアブルは11名。
- 主要クラスは1つ、役割タグは複数。
- 肖像と戦闘スプライトで顔、髪、体格、服、武器を一致。
- 同じ顔、身体、武器を色替えだけで別人物へ流用しない。
- 既存パイセン素材は非破壊で保全。
- 固有特性は自動発動または常時効果。個別アクティブスキルは0.8.0以降。
- 戦闘中barkは、該当人物が実際に出撃中の場合だけ再生。
- 人物へゲーム仕様を説明させない。

## 3. クラスID

- `class-frontline`：前衛
- `class-skirmisher`：遊撃
- `class-marksman`：射撃
- `class-heavy`：重装
- `class-support`：支援
- `class-engineer`：工兵
- `class-berserker`：狂戦

## 6. 加入・所有

- 初期所有：パイセン、ハチ、ミズチ、センセイ
- Stage 1後：タタラとクレイジーキングが調達可能
- Stage 2後：クマバーソン、ババヤガ、ロッカが調達可能
- Stage 4後：ガンテツは物語加入で無料所有
- Stage 5後：モンキーの情報判明
- Stage 6後：モンキーが調達可能
- 0.6.5で旧対応人物が解放済みの場合、schema v5 migrationで対応新人物を所有済みにする
- 既存プレイヤーから使用可能人物を取り上げない

## 7. アート制作・受入

- 差し替え5名、新規2名、ノイズは全員固有の正面・全身基準アートを作る
- 肖像と戦闘スプライトの顔、髪、体格、服、武器を一致
- 同じ顔、手、身体、武器の流用禁止
- スマートフォン戦闘画面で輪郭だけでも識別可能
- 正式画像の生成、選定、採用はChatGPT／プロデューサー側
- Codexは承認済み素材を統合し、不足素材を仮画像で完成扱いにしない
