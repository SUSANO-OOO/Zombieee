# 西新世紀末物語 — 0.7.0キャラクター正本

更新日：2026-07-17

## 1. 目的と優先順位

この文書は0.7.0のプレイアブル11名と非戦闘オペレーター1名について、内部ID、基本属性、武器、戦闘役割、口調、加入条件、アート方針を固定する。

参照優先順位：

1. `docs/PRODUCER_DECISIONS_0.7.0.md`
2. `docs/IMPLEMENTATION_DIRECTIVE_0.7.0.md`
3. 本書

主人公はプレイヤーであり、11名のいずれかではない。

## 2. 共通原則

- プレイアブルは11名。
- 主要クラスは1つ、役割タグは複数。
- 固有特性は自動発動または常時効果。個別アクティブスキルは0.8.0以降。
- 戦闘中barkは、該当人物が出撃中の場合だけ表示する。
- 人物へゲーム仕様を説明させない。
- ストーリー会話の読み上げボイスは実装しない。既存の戦闘中キャラクターボイスは維持する。
- 肖像、会話立ち絵、戦闘スプライトで顔、髪、体格、服、武器を一致させる。
- 同じ顔、身体、手、武器を色替えだけで別人物へ流用しない。
- 制作基準方向は右向き。非対称装備が反転で破綻する場合は左右専用フレームを作る。
- 同じキャラクターを、コストと再出撃待ちに従って戦場へ複数体同時召喚できる。
- 編成カード自体は人物ごとに1種類とし、7枠内の同一カード重複登録はしない。

## 3. アート責任と品質ゲート

Codexが未確定キャラクターのデザイン制作、アセット生成、加工、透過、crop、atlas、manifest、描画統合、スマートフォン受入まで担当する。

すべての新規画像は、`docs/PRODUCER_DECISIONS_0.7.0.md`で委任された自動採否、identity lock、実画面品質ゲートを通過したものだけを採用する。

### 3.1 保全対象

次の4名は現在採用済みの顔、髪、体格、衣服、武器、主要配色、人物像、口調を別物へ変更しない。

- パイセン
- クマバーソン
- ババヤガ
- クレイジーキング

### 3.2 全面再設計対象

次は既存画像を正式デザインとして保全しない。

- ハチ
- ミズチ
- ナオ
- タタラ
- レイダー
- ガンテツ
- モンキー
- いくらちゃん

人物ごとに顔型、目、眉、鼻、口、顎、年齢感、髪型、髪色、身長、体格、姿勢、主配色、武器、装備輪郭を変える。

フォトリアル、実写風、プラスチック質感の3D、幼い萌え系、均一なAI美男美女は禁止する。

- `combatKind`：`brawler`
- 主要クラス：前衛
- 役割：近接、連撃、押し返し、低コスト
- 年齢：24歳
- 発生前：西新周辺の大学生。大衆居酒屋「くまや」でアルバイト
- 武器：素手
- 外見：現行採用素材を非破壊で保全

- 口調：必要な場面で`〜っす`。語録由来の言葉は出撃・危機など決め場面だけ
- 固有特性`ノッてきたっす`：同じ対象への連続命中で短時間、攻撃速度と押し返し上昇

### 5.4 ナオ

- `unitId`：`unit-nao`
- migration alias：`unit-sensei`、旧`medic`、白石直人
- `combatKind`：`medic`
- 主要クラス：支援
- 役割：低コスト回復、援護射撃、低HP優先、救助、損害抑制
- 年齢：32歳
- 性別：女性
- 発生前：夜間動物病院の看護助手。災害後は人間も診る
- 武器：短銃身カービン、救急バッグ、止血器具
- 性格：口は悪いが患者を見捨てない。英雄的な無茶を嫌う
- 口調：診断のように短い。怒る時ほど短く、疲れている時ほど優しい
- 配置コスト初期基準：`35`
- 配置コスト調整許容範囲：`32〜38`
- 数値方針：低HP、低攻撃力。安く出せる代わりに単独で前線を維持できない
- 治療AI：最低HP比率の味方を優先する。治療対象が有効範囲外なら再索敵する
- 重複治療：同一対象へ複数のナオが集中した場合、二体目以降の回復量を段階的に減衰
- 固有特性`生きて帰る方を選べ`：最低HP比率の味方を優先治療し、治療後に短い被ダメージ軽減
- 役割分離：簡易救護所は定点範囲回復、ナオは移動単体回復
- 表示名：`ナオ`。苗字を表示しない
- デザイン目的：医療、救助、損害抑制が一目で分かり、白衣コスプレや未来医療へ寄せない。成人女性として固有の顔、髪、体格を設計する

## 7. 加入・所有

- 初期所有：パイセン、ハチ、ミズチ、ナオ、クマバーソン、ババヤガ
- Stage 1後：クレイジーキングは物語加入で無料所有。タタラは情報判明・調達可能
- Stage 2後：レイダーは情報判明・調達可能
- Stage 4後：ガンテツは物語加入で無料所有
- Stage 5：モンキーと直接合流し、Stage 6まで固定NPCとして共闘
- Stage 6後：モンキーは物語加入で無料所有
- 物語で直接仲間になる人物は無料。情報だけ判明した人物はキャップで調達
- 0.6.5で旧対応人物が使用可能だった場合、schema v5 migrationで対応人物を所有済みにする
- 既存プレイヤーから使用可能人物を取り上げない
