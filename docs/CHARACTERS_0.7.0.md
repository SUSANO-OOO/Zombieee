# 西新世紀末物語 — 0.7.0キャラクター正本

更新日：2026-07-17

## 1. 目的と優先順位

この文書は0.7.0のプレイアブル11名と非戦闘オペレーター1名について、内部ID、人物像、武器、戦闘役割、口調、加入条件、アート方針を固定する。

最上位正本は`docs/IMPLEMENTATION_DIRECTIVE_0.7.0.md`。本書と旧Issue・旧PRコメント・旧台本が矛盾する場合は実装統合指示を優先する。

主人公はプレイヤーであり、11名のいずれかではない。

## 2. 共通原則

- プレイアブルは11名。
- 主要クラスは1つ、役割タグは複数。
- 固有特性は自動発動または常時効果。個別アクティブスキルは0.8.0以降。
- 戦闘中barkは、該当人物が実際に出撃中の場合だけ表示する。
- 人物へゲーム仕様を説明させない。
- キャラクターボイスは実装しない。
- 肖像、会話立ち絵、戦闘スプライトで顔、髪、体格、服、武器を一致させる。
- 同じ顔、身体、手、武器を色替えだけで別人物へ流用しない。
- 制作基準方向は右向き。非対称装備が反転で破綻する場合は左右専用フレームを作る。

## 3. アート責任

Codexが未確定キャラクターのデザイン制作、アセット生成、加工、透過、crop、atlas、manifest、描画統合、スマートフォン受入まで担当する。

### 3.1 保全対象

次の4名は現在採用済みの顔、髪、体格、衣服、武器、主要配色を別人へ変更しない。

- パイセン
- クマバーソン
- ババヤガ
- クレイジーキング

### 3.2 全面再設計対象

次は既存画像を正式デザインとして保全しない。

- ハチ
- ミズチ
- 医療支援ユニット
- タタラ
- レイダー
- ガンテツ
- モンキー
- ノイズ

人物ごとに顔型、目、眉、鼻、口、顎、年齢感、髪型、髪色、身長、体格、姿勢、主配色、武器、装備の輪郭を変える。フォトリアル、実写風、プラスチック質感の3D、幼い萌え系、均一なAI美男美女は禁止する。

## 4. クラスID

- `class-frontline`：前衛
- `class-skirmisher`：遊撃
- `class-marksman`：射撃
- `class-heavy`：重装
- `class-support`：支援
- `class-engineer`：工兵
- `class-berserker`：狂戦

## 5. プレイアブル

### 5.1 パイセン

- `unitId`：`unit-paisen`
- `combatKind`：`brawler`
- 主要クラス：前衛
- 役割：近接、連撃、押し返し、低コスト
- 年齢：24歳
- 発生前：西新周辺の大学生。大衆居酒屋「くまや」でアルバイト
- 武器：素手
- 外見：現行採用素材を非破壊で保全
- 性格：怖くても先に身体が動く。無茶を格好いいとは思っていない
- 口調：必要な場面で`〜っす`。語録由来の言葉は出撃・危機など決め場面だけ
- 固有特性`ノッてきたっす`：同じ対象への連続命中で短時間、攻撃速度と押し返し上昇

### 5.2 ハチ

- `unitId`：`unit-hachi`
- migration alias：旧`scout`、橘迅
- `combatKind`：`scout`
- 主要クラス：遊撃
- 役割：高機動、近接、レーン移動、対高速
- 年齢：23歳
- 発生前：自転車便、夜間配送
- 武器：短く切ったバール、折り畳み式打撃棒
- 性格：道、距離、出口を常に数える。危険へ入るのは帰り道を作るため
- 口調：短文。急かすが、弱っている相手には声を落とす
- 固有特性`抜け道`：高速敵・後方侵入敵への初接敵まで移動速度上昇。最初の一撃に短い足止め
- デザイン目的：小柄、高機動、軽装、帰還路を作る遊撃手だと縮小画面で分かる

### 5.3 ミズチ

- `unitId`：`unit-mizuchi`
- migration alias：旧`ranger`、黒木凛
- `combatKind`：`ranger`
- 主要クラス：射撃
- 役割：遠距離、精密、対特殊、初弾強化
- 年齢：29歳
- 発生前：害獣駆除会社の現場責任者
- 武器：半自動小銃、簡易消音器、低倍率光学照準器
- 性格：静かだが無感情ではない。呼吸、反響、動線を見る
- 口調：断定が多く、長い説明を嫌う
- 固有特性`一発目を外さない`：新しい特殊個体への初弾に追加ダメージと短いマーキング
- デザイン目的：長距離精密射撃と特殊個体観察が一目で分かる

### 5.4 医療支援ユニット

- target `unitId`：`unit-medic`
- migration alias：`unit-sensei`、旧`medic`、白石直人
- `combatKind`：`medic`
- 主要クラス：支援
- 役割：回復、援護射撃、低HP優先、損害抑制
- 年齢：32歳
- 発生前：夜間動物病院の看護助手。災害後は人間も診る
- 武器：短銃身カービン、救急バッグ、止血器具
- 性格：口は悪いが患者を見捨てない。英雄的な無茶を嫌う
- 口調：診断のように短い。怒る時ほど短く、疲れている時ほど優しい
- 固有特性`生きて帰る方を選べ`：最低HP比率の味方を優先治療し、短い被ダメージ軽減
- 表示名：`センセイ`は不採用の仮称。Codexが正式名を決定し、理由を本書へ記録する
- デザイン目的：医療、救助、損害抑制が一目で分かり、白衣コスプレや未来医療へ寄せない

### 5.5 タタラ

- `unitId`：`unit-tatara`
- migration alias：旧`brute`、大庭豪
- `combatKind`：`brute`
- 主要クラス：重装
- 役割：破砕、対装甲、対拠点、怯み
- 年齢：38歳
- 発生前：解体工事、鉄骨切断、夜間道路工事
- 武器：短柄の大型解体ハンマー
- 性格：普段は穏やか。壊す前に構造を見る
- 口調：ゆっくり。余計な比喩を使わない
- 固有特性`壊れる場所は決まっている`：重装敵、前面装甲、感染拠点へ高ダメージ。連続攻撃で強い怯みまたは装甲破損
- デザイン目的：工事現場の職人と破砕役が分かる。ファンタジー戦士にしない

### 5.6 クレイジーキング

- `unitId`：`unit-crazy-king`
- `combatKind`：`crazy-king`
- 主要クラス：狂戦
- 役割：近接、範囲、対群体、対拠点
- 年齢：不詳。成人
- 武器：チェーンソー
- 外見：現行の黄色い筒状頭部、緑のパーカー、赤いブーツを維持
- 性格：イカれて見えるが、燃料、退路、罠、生存者の位置を正確に把握
- 口調：漢字とカタカナ中心。原則二言三言。長台詞、丁寧な説明、連続した冗談は禁止
- 固有特性`止めたら死ぬ`：連続命中で回転数、攻撃範囲、対群体性能が段階上昇

### 5.7 クマバーソン

- `unitId`：`unit-kumaverson`
- `combatKind`：`kumaverson`
- 主要クラス：前衛
- 役割：近接、足止め、味方保護、自己回復
- 年齢：成人
- 発生前：大衆居酒屋「くまや」を切り盛り。TAKUYAは常連客で友人
- 武器：フライパン
- 外見：現行の黒髪、汚れた白Tシャツ、黒パンツを維持
- 性格：熱血パワー型。食事と睡眠を人間が生きる最低条件として扱う
- 口調：博多弁。大声が多いが、子どもや負傷者には声を落とす
- 固有特性`腹減っとるやろ`：近い味方が危険HPになると、出撃ごとに一度だけ携行食で小回復と短い怯み耐性

### 5.8 ババヤガ

- `unitId`：`unit-babayaga`
- `combatKind`：`babayaga`
- 主要クラス：射撃
- 役割：中遠距離、精密、特殊優先、弱点攻撃
- 年齢：27歳
- 発生前：大手証券会社社員。発生当夜は大衆居酒屋「くまや」の客
- 武器：サプレッサー付き拳銃
- 外見：現行の黒髪、白いワイシャツ、ネクタイ、ショルダーホルスターを維持
- 性格：基本は冷静で数字に強い。既婚で妻に頭が上がらない
- 口調：通常は標準語。焦り、怒り、家庭の話では博多弁。奇妙な比喩は一場面に一つ以下
- 固有特性`数字が合わん`：特殊個体の弱点露出時、短時間だけ射程とクリティカル率上昇

### 5.9 レイダー

- `unitId`：`unit-raider`
- migration alias：`unit-rokka`、旧`gunner`、真壁玲奈
- `combatKind`：`gunner`
- 主要クラス：射撃
- 役割：制圧、連射、対群体、中遠距離
- 年齢：27歳
- 発生前：ライブハウスの音響担当
- 武器：軽機関銃
- 性格：騒音にうるさく、自分の銃声にも文句を言う。戦闘を拍子で捉える
- 口調：乾いた冗談。短いカウント
- 固有特性`四つ打ち`：同じレーンへの連射で制圧値上昇、敵前進速度低下。継続で過熱
- 表示名：`ロッカ`は不採用。正式表示名は`レイダー`
- デザイン目的：音響担当の実務感と制圧射撃役を両立し、歌手・アイドル・サイバーパンクにしない

### 5.10 ガンテツ

- `unitId`：`unit-gantetsu`
- `combatKind`：`guardian`
- 主要クラス：重装
- 役割：盾、防御、味方保護、足止め
- 年齢：42歳
- 発生前：西新駅の警備員。地域消防団経験
- 関係：モンキーと11年来の仕事仲間
- 武器：警棒、短い散弾銃、ホーム柵と防火板を溶接した大型盾
- 性格：極端に礼儀正しく、生存者の人数と位置を忘れない。自分を人数へ入れない癖がある
- 口調：丁寧語。危険時ほど静か
- 固有特性`ここから先は通しません`：後方味方への近接攻撃の一部を肩代わりし、一定時間踏みとどまる
- デザイン目的：地域の警備員、防御役、大型盾を明確にし、警察特殊部隊や騎士にしない

### 5.11 モンキー

- `unitId`：`unit-monkey`
- `combatKind`：`engineer`
- 主要クラス：工兵
- 役割：設置、妨害、足止め、対地形
- 年齢：26歳
- 発生前：地下鉄の保線・設備点検作業員
- 関係：ガンテツと11年来の仕事仲間
- 武器：工業用釘打ち機、レンチ、携帯工具
- 性格：設備には丁寧、人間には正確さを求める。設備を守ることが人命を守ると理解している
- 口調：独り言が多い。機械へ話す時だけ丁寧
- 固有特性`そこ、踏むなよ`：一定間隔で担当レーン後方へ一回使用の自動足止め装置を設置
- デザイン目的：保線作業員と工兵が分かる。猿の耳、尻尾、動物化は禁止

## 7. 加入・所有

- 初期所有：パイセン、ハチ、ミズチ、医療支援ユニット、クマバーソン、ババヤガ
- Stage 1後：クレイジーキングは物語加入で無料所有。タタラは情報判明・調達可能
- Stage 2後：レイダーは情報判明・調達可能
- Stage 4後：ガンテツは物語加入で無料所有
- Stage 5：モンキーと直接合流し、Stage 6まで共闘
- Stage 6後：モンキーは物語加入で無料所有
- 物語で直接仲間になる人物は無料。情報だけ判明した人物はキャップで調達
- 0.6.5で旧対応人物が使用可能だった場合、schema v5 migrationで対応人物を所有済みにする
- 既存プレイヤーから使用可能人物を取り上げない
